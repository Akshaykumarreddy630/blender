# 🧱 Stack Implementation in C++ (Blender Assignment)

This project demonstrates a basic **stack implementation in C++**, packaged using **CMake**, with associated **UML diagrams** and UI behavior captured in a screenshot. Created as part of an assignment.

---

## 📂 Files Overview

- `stack.cc` – C++ source file with stack implementation logic
- `CMakeLists.txt` – Configuration file for compiling using CMake
- `class diagram.uxf` – UML Class Diagram (use UMLet or compatible viewer)
- `sequence diagram.uxf` – UML Sequence Diagram
- `Screenshot (59).png` – Output screenshot/UI visual (if applicable)

---

## ⚙️ Build Instructions (using CMake)

```bash
mkdir build
cd build
cmake ..
make
./your_program_name
````
